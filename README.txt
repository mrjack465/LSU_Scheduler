Site will have to be run on localhost using an apache server. MySQL is also required
for database support. 

1. Set up Apache settings and set DocumentRoot location to 'LSU_Scheduler'
2. Set up a new database called 'LSU_Scheduler' in MySQL
3. Import 'users.sql','professors.sql', and 'courses.sql' into the LSU_Scheduler 
database. Files are located in the 'LSU_Scheduler' directory
4. Start Apache server and MySQL
5. Open http://localhost/, or http://127.0.0.1; This may vary depending on
system setup.
